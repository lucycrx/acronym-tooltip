Privacy Policy

Last updated: February 16, 2026

Acronym Tooltip ("the Extension") is a Chrome browser extension that displays definitions for workplace acronyms on Meta internal sites. This policy explains what data the Extension accesses, how it is used, and what is stored.

Scope

The Extension activates only on the following domains:

  • *.workplace.com
  • *.internalfb.com
  • *.facebook.com
  • *.meta.com
  • docs.google.com

It does not run on any other websites.

Data Collected

The Extension does not collect, transmit, or store any personal data. Specifically:

  • Page content: The Extension reads text on the current page solely to identify acronyms and provide surrounding context for AI-generated definitions. This text is processed locally or sent to Meta-internal APIs only. It is never sent to third-party services.
  • Acronym lookups: When you hover over an acronym, the Extension queries Meta's internal WUT API to retrieve definitions. If no WUT definition exists and AI fallback is enabled, it sends the acronym and a short surrounding text snippet to a Meta-internal AI service.
  • User preferences: Settings such as dismissed acronyms, scan frequency, and feature toggles are stored locally in your browser using Chrome's chrome.storage API. This data never leaves your device.

Data Sharing

The Extension does not share data with any third parties. All network requests are made exclusively to Meta-internal endpoints.

Data Storage

All user preferences and dismissed-acronym lists are stored locally in your browser via chrome.storage.local. No data is stored on external servers by the Extension.

Permissions

The Extension requests the following browser permissions:

  • storage: To save your preferences (dismissed terms, settings) locally.
  • activeTab: To access the content of the current tab for acronym detection.
  • Host permissions: To make API requests to Meta-internal services (WUT, AI fallback) and to run content scripts on supported domains.

Changes to This Policy

If this policy is updated, the "Last updated" date above will be revised. Continued use of the Extension after changes constitutes acceptance of the updated policy.

Contact

For questions about this privacy policy or the Extension, please file an issue on the project's repository or reach out to the extension maintainer.
